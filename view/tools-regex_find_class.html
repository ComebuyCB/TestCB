在HTML或jQuery中，用正規表達式尋找className
純 class：
尋找： class=(["'])([^"']*\s+)?()(\s+[^"']*)?(["'])
取代： class=$1$2$4$5
純 jquery：
尋找： \((\s*["']\s*)([\.])?([^"']*[^\w-])?()([^\w-][^"']*)?(\s*["']\s*)\)
取代： ($1$2$3$5$6)
class + jquery：
尋找： class=(["'])([^"']*\s+)?()(\s+[^"']*)?(["'])|\((\s*["']\s*)([\.])?([^"']*[^\w-])?()([^\w-][^"']*)?(\s*["']\s*)\)
