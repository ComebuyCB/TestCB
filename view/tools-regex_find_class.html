用正規表達式找尋想要的class

* 純 class:
class\s*=\s*["']([^"']*\s+)?(\s+[^"']*)?["']
* 純 jquery：
\(\s*["']\s*[\.]?([^"']*[^\w-])?([^\w-][^"']*)?\s*["']\s*\)
* class + jquery:
class\s*=\s*["']([^"']*\s+)?(\s+[^"']*)?["']|\(\s*["']\s*[\.]?([^"']*[^\w-])?([^\w-][^"']*)?\s*["']\s*\)
匹配範例：
* class:
能匹配：class="" | class=" xxx" | class="xxx " | class="xxx yyy" | class = " "
不匹配：class="__" | class="__" | class="x" type=""
* jquery:
能匹配：("") | (".") | ( " " ) | ("{},[],") | ("xxx+.>yyy")
不匹配：(".__") | (".__")
可至： https://regex101.com 試試
