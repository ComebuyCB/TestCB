CSS Selector[CSS選擇器]

            [target = _blank] 完全一樣
            [title ~= flower] 單字一樣
            [lang  |= en]	起始單字一樣，後面加 - 才可接字
            [href  ^= "https"] 起始一樣
            [href  $= ".pdf"] 結尾一樣
            [href  *= "w3schools"] 搜到一樣

            div + p|父.父|排在div後面的下一位p
            div ~ p|父.父|排在div後面的所有p

VB語言

            <script runat="server">
            Public function rrq(vStr)
            return replace(request(vStr),"'","''")
            End function
            </script>
            '用來取url?後的敘述。如:XX.html?du=21
            '那麼下 rrq('du') 即可獲取'21'這個值
